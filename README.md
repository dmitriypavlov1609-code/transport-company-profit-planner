# Transport Company Profit Planner

Веб-калькулятор прибыли для транспортной компании.

## Функции

- расчет выручки, расходов, чистой прибыли и маржи
- отдельная страница отчета `report.html` с красивым русским оформлением
- P&L (выручка, себестоимость, валовая прибыль, OPEX, EBITDA, налоги, чистая прибыль)
- ДДС (денежные поступления/выплаты, чистый денежный поток)
- метрики: комиссия агрегатора, зарплаты водителей и офиса, страхование, софт, лизинг, налоги
- логика классификации:
  - себестоимость = топливо + зарплата водителей + комиссия агрегатора
  - операционные расходы = прочие расходы (кроме налогов и амортизации)
- точка безубыточности по количеству рейсов
- сценарии: пессимистичный / базовый / оптимистичный
- скачивание отчета в форматах: PDF, CSV, JSON, TXT

## Локальный запуск

```bash
cd transport-company-profit-planner
python3 -m http.server 8080
```

Откройте: `http://localhost:8080`
